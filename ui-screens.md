# UI 页面设计

> 每个页面的详细描述，包含布局、交互、状态

---

## 一、导航架构

```
iOS (TabView):
  📋 案件列表    👥 人脉列表    📅 日历    ⚙️ 设置
      │              │             │
  案件详情页    人脉详情页    事件列表/创建
      │              │
  案件编辑页    人脉编辑页
      │
  添加参与人弹窗 / 新建联系人弹窗
```

```
iPad / Mac (NavigationSplitView):
  侧栏: 案件 | 人脉 | 日历 | 设置
  内容区: 列表（左） + 详情（右）
```

## 二、案件列表页

### 布局
```
┌────────────────────────────────┐
│  🔍 搜索案件...          [+ 新建] │  ← 搜索栏 + 新建按钮
├────────────────────────────────┤
│  [全部] [进行中] [已结案] [刑事] [民事]...│  ← 快速筛选分段控件
├────────────────────────────────┤
│  📋 XX公司股权转让纠纷案          │
│     (2026)京0105民初12345号     │
│     🏛 朝阳法院 · ⚖️ 审理中      │
│     👤 5人参与 · 📅 7/15开庭     │  ← 案件卡片
├────────────────────────────────┤
│  📋 XX公司合同诈骗案              │
│     (2025)京0105刑初6789号      │
│     🏛 朝阳法院 · ✅ 已判决       │
│     👤 4人参与                   │
├────────────────────────────────┤
│  ...更多案件...                  │
└────────────────────────────────┘
```

### 交互
- 下拉刷新
- 点击案件 → push 案件详情
- 长按 → 快捷操作（归档/编辑/删除）
- 搜索实时过滤
- 空状态：插画 + "添加你的第一个案件"提示

## 四、添加参与人弹窗（最高频交互）

### 布局
```
┌────────────────────────────────┐
│  添加参与人            [取消]    │
├────────────────────────────────┤
│  🔍 输入姓名搜索已有联系人...    │  ← 实时搜索
├────────────────────────────────┤
│  搜索结果:                      │
│  ┌──────────────────────────┐ │
│  │ 🧑 王强                    │ │  ← 已有联系人，点击选择
│  │   当事人 · XX公司法人代表    │ │
│  ├──────────────────────────┤ │
│  │ 🔨 张建国  法官·朝阳法院    │ │
│  ├──────────────────────────┤ │
│  │ 👩‍💼 刘芳  律师·君合所       │ │
│  ├──────────────────────────┤ │
│  │ [+ 新建联系人 "王伟"]      │ │  ← 搜不到？快捷新建
│  └──────────────────────────┘ │
│                                │
│  --- 或 ---                    │
│  浏览全部联系人...              │  ← 进入完整人脉列表选择
└────────────────────────────────┘

选中某人后 →
┌────────────────────────────────┐
│  添加参与人                     │
│  已选择: 王强                  │
│                                │
│  在本案角色:                    │
│  [当事人 ▼]                    │  ← 下拉选择或自定义输入
│                                │
│  角色大类:                      │
│  [当事人] [律师] [法官] [其他]   │
│                                │
│  备注（可选）:                  │
│  ┌──────────────────────────┐ │
│  │                           │ │
│  └──────────────────────────┘ │
│                                │
│  [确认添加]                    │
└────────────────────────────────┘
```

### 交互细节
- 搜索框获得焦点时自动弹出键盘
- 输入 1-2 个字即开始搜索（防抖 200ms）
- 搜索结果按匹配度和重要性排序
- 支持拼音搜索（"zs" 匹配 "张三"）
- "新建联系人"入口始终在最底部

## 十二、设计系统（简要）

### 色彩
- 主色：专业蓝 `#1565C0`（法律专业感）
- 法官红 `#D32F2F`（角色标签色）
- 检察官蓝 `#1976D2`
- 律师绿 `#388E3C`
- 成功/结案绿 `#4CAF50`
- 警告/进行中橙 `#FF9800`
- 背景色：系统背景（支持深色模式）

### 图标
- 优先 SF Symbols（Apple 内建，6000+ 图标免费）
- App Icon：⚖️ 天平 + 关联线

### 字体
- 系统字体（San Francisco），不做自定义
- 中文：PingFang SC
- 等宽数字：`.monospacedDigit()`
