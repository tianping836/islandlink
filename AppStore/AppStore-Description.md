# App Store 描述

## 中文版（中国区）

### 名称
CaseNetwork — 律师案件人脉管理

### 副标题
案件与⼈脉，⼀⽹打尽

### 描述

CaseNetwork 是专为律师打造的案件与人脉交叉管理工具。

**核心能力：**

搜一个案件，看到全部参与人——法官、检察官、对方律师、当事人，以及每个人在案中扮演的角色。搜一个人，看到他的全部关联案件、所在机构、介绍人链条。三个维度自由跳转，信息不再割裂。

**为什么律师需要 CaseNetwork？**

传统通讯录和案件管理系统是两套独立的工具。当事人在微信里，法官电话在备忘录里，案件进展在脑子里。CaseNetwork 把它们编织成一张网——人、案、机构，三维交叉关联，任何一点进去都能看到全局。

**功能亮点：**

- 🔗 三维关联搜索：搜案件 → 全部参与人及角色；搜人名 → 全部关联案件及机构
- 📋 案件 Pipeline 管理：从洽谈、委托、立案到结案，九阶段清晰追踪
- 👥 人脉图谱：介绍人链、关系阶段、重要度评分、人情往来记录
- 📅 开庭日历：月/周视图 + 十一类事件彩色标记 + 开庭提醒
- 🔐 Face ID 应用锁：律师级数据安全
- ☁️ iCloud 同步：iPhone、iPad、Mac 数据无缝同步
- 📤 数据自主权：一键导出 JSON/CSV，随时备份或迁移

**隐私优先：**

所有数据存储在你的设备上，由你掌控。不上传任何服务器，不收集任何使用数据。

CaseNetwork —— 让每一条人脉都为案件服务。

## English Version (International)

### Name
CaseNetwork — Legal Case & Contact Manager

### Subtitle
Connect every case to every contact.

### Description

CaseNetwork is a case-and-contact relationship manager built for legal professionals.

**Core Capability:**

Search a case → see every participant and their role. Search a person → see every related case, organization, and referral chain. Three dimensions (Cases, Contacts, Organizations), one interconnected network.

**Why CaseNetwork?**

Traditional contact books and case management software live in separate worlds. Your clients are in WhatsApp, judges' numbers are in Notes, case progress lives in your head. CaseNetwork weaves them into a single fabric — every entry point reveals the full picture.

**Key Features:**

- 🔗 3D Cross-Reference Search: Case → all participants with roles; Person → all cases + organizations
- 📋 Case Pipeline: From consultation to closure, 9-stage visual tracking
- 👥 Contact Graph: Referral chains, relationship stages, importance scoring, interaction history
- 📅 Court Calendar: Month/week views, 11 event types color-coded, hearing reminders
- 🔐 Biometric Lock: Face ID / Touch ID for attorney-grade data protection
- ☁️ iCloud Sync: Seamless sync across iPhone, iPad, and Mac
- 📤 Data Portability: One-tap export to JSON/CSV. Your data, your control.

**Privacy First:**

All data stays on your device. No servers. No analytics. No tracking.

CaseNetwork — Every connection serves the case.
